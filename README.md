Calendar implement by C Language
================================

## Clone Repository
    $ git clone git@github.com:taewonkim/calendar.git ./calendar

*or*

    $ git clone https://github.com/taewonkim/calendar.git ./calendar
    
## Compile  
### Release  
    $ cd ./calendar
    $ gcc -o cal ./src/*.c

### Debug
    $ cd ./calendar
    $ gcc -g -Wall -o cal ./src/*.c
